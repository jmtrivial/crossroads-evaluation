Crossroad quality evaluation

This evaluation toolkit has been developped within the ANR ACTIVmap project.

Crossroad segmentation

  1. Compute crossroad segmentation using cseg. Example (compute all crossroads at distance less than 1000 meters from the obélisque (Clermont-Ferrand):

    git clone https://gitlab.limos.fr/jmafavre/crossroads-segmentation
cd crossroads-segmentation/src
./get-crossroad-description.py --by-name obélisque -r 1000 --connection-intensity 4 --to-json-all crossroads-obelisque-1000.json
  2. Follow the link for segmentation evaluation, load crossroads-obelisque-1000.json and answer all the questions. Download the updated json file with the evaluation answers.
  3. Use the evaluation browser to have a user friendly rendering of this json file. (available soon)
